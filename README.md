# Fixel Web Application

## Overview

Fixel is a web-based image processing application developed using **Streamlit** and **OpenCV**. The platform enables users to upload or capture images and apply various image enhancement and transformation techniques through an intuitive and interactive interface.

The project was developed as a practical application of digital image processing concepts, providing users with real-time visual feedback while exploring different enhancement methods.

---

## Features

* Upload images from local storage
* Capture images using a webcam
* Real-time image enhancement and transformation
* User-friendly Streamlit interface
* Educational explanations of image processing techniques

---

## Image Processing Techniques

### Thresholding

Converts grayscale images into binary images based on a selected threshold value.

### Linear Negative Transformation

Creates a negative version of an image by inverting pixel intensities.

### Power-Law (Gamma) Transformation

Enhances image contrast using gamma correction for improved visualization.

### Morphological Operations

Performs shape-based image transformations including:

* Erosion
* Dilation
* Opening
* Closing

## How It Works

1. Upload an image or capture one using your webcam.
2. Select the desired image processing technique.
3. Adjust parameters if needed.
4. View the processed image instantly.
5. Compare results and explore different enhancement methods.
